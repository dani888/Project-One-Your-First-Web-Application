Welcome to Wars Card Game!

Wars Card Game Guide:

1.) The game starts off with a shuffled deck of 52 cards.

2.) To start the game press the Start New Game button!

3.) There are 26 card draws in a set using the Draw Button.

You can now draw cards!


4.) To earn points you hopefully will get a higher card value then the computer.

5.) If You and the computer get the same card value you will draw another set of cards and whoever wins the round after the draw will get 3 Points!

6.) You and the Computer will have 20 Health Points at the start of the game. At the end of each set of cards the difference in points will determine how much health is lost for the person who lost the round.

7.) The Goal is to get the computers health points to 0 to Win the game.

Good Luck!
